# Student Submission Form (with Sl No. and Custom Headers)

## Google Sheet Setup

1. Name the sheet: `FormResponses`
2. In the first row, enter these headers:
   Sl No. | Name | Age | Class | E-mail | Phone No. | City | Timestamp
